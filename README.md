# blog-symfony
Blog app with Symfony and Docker. Made to learn.



*Developpement in progress ...*





### Requirements

- Docker
- Docker-compose

### Installation 

```bash
  docker-compose build
  docker-compose up -d
```

*The application is available at http://127.0.0.1:9000*
